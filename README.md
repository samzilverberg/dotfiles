
# Sam Zilverberg dotfiles !

## usage

```
# clone repo
cd ~
git clone https://github.com/samzilverberg/dotfiles.git
cd dotfiles
cd [macos | arch]
sudo ./install.sh
```

## supported os

- macos
- linux arch based (manjaro)

## whats config dir?

holds shared config that all OS can use
